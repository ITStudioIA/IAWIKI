# QQ机器人

## 结构设计
采用capcat + nonebot架构搭建QQ机器人

## capcat是什么？
基于 TypeScript 构建的 Bot 框架，通过相应的启动器或者框架，主动调用 QQ Node 模块提供给客户端的接口，实现 Bot 的功能。
### capcat官方文档
https://napneko.github.io/

## Nonebot
NoneBot2 是一个现代、跨平台、可扩展的 Python 聊天机器人框架（下称 NoneBot），它基于 Python 的类型注解和异步优先特性（兼容同步），能够为你的需求实现提供便捷灵活的支持。同时，NoneBot 拥有大量的开发者为其开发插件，用户无需编写任何代码，仅需完成环境配置及插件安装，就可以正常使用 NoneBot。
### Nonebot官方文档
https://nonebot.dev/docs/
